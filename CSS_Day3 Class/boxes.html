margin :

Font size controls how big or small text appears (height/dimension), measured in pixels or points, while font weight controls the thickness or boldness of the letter strokes (e.g., light, normal, bold, black) using numerical values (100-900) or keywords, affecting emphasis and visual hierarchy rather than physical dimensions.
